Element

sealed class Element

Base sealed class for all drawable elements on the canvas.

The Element type system allows the renderer to handle different drawing primitives uniformly. Use pattern matching or when expressions to handle different element types.

abstract val createdAt: Long

Epoch milliseconds at which this element was first created. Used for ordered replay (and later as a tie-break source for collab clocks). 0L means "unknown" — element predates the timestamp field; consumers that sort by createdAt should treat 0 as "earliest".
