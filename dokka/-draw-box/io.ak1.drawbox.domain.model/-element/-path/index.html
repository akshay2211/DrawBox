Path

data class Path(val id: String = Uuid.random().toString(), val type: String = "Path", val points: List<Offset>, val strokeColor: Color, val strokeWidth: Float, val alpha: Float, val zIndex: Int = 0, val rotation: Float = 0.0f, val strokeStyle: StrokeStyle = StrokeStyle.SOLID) : Element

Represents a freehand drawn path created in PEN mode.

A path is a series of connected points drawn by the user. It's rendered as a continuous curve with the specified stroke properties.
