Viewport

data class Viewport(val offset: Offset = Offset.Zero, val scale: Float = 1.0f)

Camera transform for the infinite canvas. Elements are stored in world coordinates; the viewport projects them onto the screen using a translate

  • uniform-scale transform.

Screen-pixel point s and world point w relate as: s = w * scale + offset w = (s - offset) / scale

Camera state is session state — not serialized, not in undo history. Mutate by returning a new Viewport via panBy, zoomBy, or zoomTo.
