# Drawing Tools

Learn about DrawBox's drawing tools and features.

## Core Tools

DrawBox provides intuitive tools for creating drawings:

- **Pen** - Freehand drawing with smooth paths
- **Shapes** - Rectangle, Circle, Triangle, Arrow, Line
- **Color Control** - Set stroke and background colors
- **Stroke Width** - Adjustable line thickness
- **Opacity** - Transparency control
- **History** - Unlimited undo/redo

See [Drawing Modes](drawing-modes.md) for detailed information.
